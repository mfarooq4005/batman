Step 2 — API call example
مثلاً آج کی attendance summary:

GET /api/attendance/summary-today
curl سے:
Bash
curl -H "X-API-Token: super-secret-api-token" \
"http://3.238.205.234:8090/api/attendance/summary-today?organization_code=MSC&campus_code=BOYS&branch_code=MAIN"
Expected response:
JSON
{
 "present": 48,
 "late": 5,
 "absent": 2
}
